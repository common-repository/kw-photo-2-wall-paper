Photo 2 wall-paper

== Description ==

This plug-in is designed to easily create and sell wall-papers from photos.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/kw-photo-2-wall-paper` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Use the WooCommerce->Settings->Wall Paper screen to configure the plugin.
1. Activate Wall Paper for each concerned product.

== Frequently Asked Questions ==

= Do I need a running copy of WooCommerce? =

Yes. This plug-in adds wall-paper ordering specific functionalities to your existing WooCommerce installation.

= Is there any standard size of wall paper?

Yes. Standard wallpaper rolls made in Europe have dimensions of 10.05 m x 0.53 m, which amounts to 5 m2 (square meters).

= Does the plug-in support imperial units?

Currently only metric units are supported.
